# 鬼畜钢琴 (Kuso Piano)

一个基于网页开发的互动性恶搞（鬼畜）音效与动效播放器。支持在电脑和移动端上进行鬼畜多屏连播。通过屏幕底部的模拟钢琴按键，点击触发对应的动效和音效！

## ✨ 核心特性

- **跨平台兼容**：适配 PC 鼠标点击、键盘快捷键，以及移动端的直接触屏点击。
- **模拟钢琴交互**：底部自带 12 个黑白钢琴键，提供真实的点击按压视觉反馈。
- **多开堆叠**：支持同屏最大同时联动播放 5 个鬼畜资源（动态排列展示）。
- **容错与静默加载**：当您尚未添加完某个琴键对应的资源时（如缺失第 7 个动效），程序会自动静默拦截，不影响页面运行且控制台不报红错，非常适合逐步添加您的素材！

## 🚀 资源部署方式

由于本项目采用前端原生的图片、音频文件加载，请按照以下规约存放你的素材：

1. 在 `assets` 文件夹下准备你的素材。
2. 每个素材对应 `1.gif` 和 `1.wav`（按键1对应编号1，以此类推，最大到12）。

## ⌨️ 按键说明

支持任意多端：
- **移动端/鼠标**：直接点击屏幕下方的钢琴键即可。
- **PC 键盘快捷键映射**：
  - 数字键区：`1`、`2`、`3`、`...`、`0`、`-`、`=` 分别对应 1-12 键。
  - 字母区：传统版 `Q`、`W`、`E`、`R` 也可以触发对应动画。

## 🛠 技术栈
- 纯 HTML5 / DOM API
- Vanilla JavaScript (ES6) 
- 纯 CSS3 (Flexbox 及过渡动画)
